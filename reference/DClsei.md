# Constrained Least Squares Deconvolution

Constrained Least Squares Deconvolution

## Usage

``` r
DClsei(b, A, G, H, scaling)
```

## Arguments

- b:

  Observation vector.

- A:

  Signature matrix.

- G:

  Constraint matrix.

- H:

  Constraint vector.

- scaling:

  Scaling factor.

## Value

Estimated cell fractions.
