# Reset IOBR Cache Directory to Default

Resets the cache directory to the default session-specific temporary
directory.

## Usage

``` r
reset_iobr_cache_dir()
```

## Value

Invisibly returns the default cache directory path.

## Examples

``` r
reset_iobr_cache_dir()
#> ✔ Cache directory reset to session-specific temporary directory: /tmp/Rtmp6CKTc5/IOBR_cache
```
